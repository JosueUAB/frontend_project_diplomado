# Frontend - Gestión de Tareas

Proyecto frontend minimal en React + Vite (TypeScript). Expone una UI básica para crear, listar y actualizar el estado de tareas.

Cómo ejecutar:

1. Entrar a la carpeta `frontend`.
2. Instalar dependencias: `npm install`.
3. Ejecutar: `npm run dev`.

La aplicación asume que la API backend está disponible en la misma origen (misma URL/puerto). Si el backend está en otro puerto, configura un proxy en Vite o usa rutas absolutas.
